5.1 Semiconductor Main Memory
🧠 แนวคิดหลัก
หน่วยความจำหลักปัจจุบันใช้ semiconductor chip
หน่วยพื้นฐานคือ memory cell
เก็บค่าได้ 2 สถานะ (0 / 1)
อ่าน (read) และเขียน (write) ได้

🔸 ประเภท RAM
| ประเภท | ลักษณะ                                 |
| ------ | -------------------------------------- |
| DRAM   | ใช้ capacitor เก็บประจุ → ต้อง refresh |
| SRAM   | ใช้ flip-flop → เร็วกว่า แต่แพง        |

🔸 เปรียบเทียบ DRAM vs SRAM
| คุณสมบัติ     | DRAM      | SRAM      |
| ------------- | --------- | --------- |
| ความเร็ว      | ช้า       | เร็ว      |
| ราคา          | ถูก       | แพง       |
| การเก็บข้อมูล | capacitor | flip-flop |
| ต้อง refresh  | ต้อง      | ไม่ต้อง   |

🔸 ประเภท ROM
| ประเภท | วิธีลบข้อมูล    | คุณสมบัติ        |
| ------ | --------------- | ---------------- |
| ROM    | ลบไม่ได้        | เขียนถาวร        |
| PROM   | เขียนครั้งเดียว | ยืดหยุ่นกว่า ROM |
| EPROM  | ใช้ UV ลบ       | ลบทั้งชิป        |
| EEPROM | ลบด้วยไฟฟ้า     | ลบเป็น byte      |
| Flash  | ลบเป็น block    | เร็วและนิยม      |
