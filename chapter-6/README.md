## วัตถุประสงค์การเรียนรู้ (Learning Objectives)

หลังจากศึกษาบทนี้แล้ว คุณควรจะสามารถ:

เข้าใจคุณสมบัติหลักของ ดิสก์แม่เหล็ก (Magnetic Disks)

เข้าใจปัญหาด้านประสิทธิภาพที่เกี่ยวข้องกับการเข้าถึงดิสก์แม่เหล็ก

อธิบายแนวคิดของ RAID และบรรยายระดับต่าง ๆ ของ RAID ได้

เปรียบเทียบความแตกต่างระหว่าง ฮาร์ดดิสก์ (HDD) และ โซลิดสเตตดิสก์ (SSD)

อธิบายหลักการทำงานโดยทั่วไปของ หน่วยความจำแฟลช (Flash Memory)

เข้าใจความแตกต่างระหว่างสื่อจัดเก็บข้อมูลแบบ ออปติคอลดิสก์ (Optical Disk) แต่ละชนิด

อธิบายภาพรวมของเทคโนโลยีการจัดเก็บข้อมูลแบบ เทปแม่เหล็ก (Magnetic Tape)

## 📚 เนื้อหาของบทนี้

บทนี้จะกล่าวถึงอุปกรณ์และระบบหน่วยความจำภายนอกหลายประเภท 

โดยเริ่มจาก:อุปกรณ์ที่สำคัญที่สุด คือ ดิสก์แม่เหล็ก (Magnetic Disk)ซึ่งเป็นพื้นฐานของหน่วยความจำภายนอกในระบบคอมพิวเตอร์เกือบทั้งหมดจากนั้นจะกล่าวถึงการใช้งาน ชุดดิสก์ (Disk Arrays)

เพื่อเพิ่มประสิทธิภาพ โดยเน้นไปที่ระบบที่เรียกว่า RAID (Redundant Array of Independent Disks)

ต่อมาจะอธิบายเกี่ยวกับโซลิดสเตตดิสก์ (SSD) ซึ่งมีความสำคัญมากขึ้นในระบบคอมพิวเตอร์ปัจจุบันหลังจากนั้นจะพูดถึง

หน่วยความจำแบบออปติคอล (Optical Memory) เช่น CD, DVDและสุดท้ายจะอธิบาย

เทคโนโลยีเทปแม่เหล็ก (Magnetic Tape)
