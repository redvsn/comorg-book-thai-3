## 7.10 คำศัพท์สำคัญ คำถามทบทวน และแบบฝึกหัด

### คำศัพท์สำคัญ

| Interrupt-Driven I/O | I/O ที่ขับเคลื่อนด้วยอินเทอร์รัปต์ |

| I/O Command | คำสั่ง I/O |

| Write Update | การอัปเดตการเขียน |

### คำถามทบทวน

**7.1** จงระบุการจำแนกประเภทกว้าง ๆ สามประเภทของอุปกรณ์ภายนอกหรืออุปกรณ์ต่อพ่วง

**7.2** International Reference Alphabet คืออะไร

**7.3** หน้าที่หลักของโมดูล I/O มีอะไรบ้าง

**7.4** จงระบุและให้นิยามโดยย่อของเทคนิคสามวิธีในการดำเนินการ I/O

**7.5** ความแตกต่างระหว่าง Memory-Mapped I/O และ Isolated I/O คืออะไร

**7.6** เมื่ออินเทอร์รัปต์อุปกรณ์เกิดขึ้น โปรเซสเซอร์จะพิจารณาได้อย่างไรว่าอุปกรณ์ใดออกอินเทอร์รัปต์

**7.7** เมื่อโมดูล DMA เข้าควบคุมบัส และในขณะที่ยังคงควบคุมบัสอยู่ โปรเซสเซอร์จะทำอะไร

### แบบฝึกหัด

**7.1** ในไมโครโปรเซสเซอร์ทั่วไป ที่อยู่ I/O เฉพาะใช้เพื่ออ้างถึงรีจิสเตอร์ข้อมูล I/O และที่อยู่เฉพาะสำหรับรีจิสเตอร์ควบคุมและสถานะในตัวควบคุม I/O สำหรับอุปกรณ์ที่กำหนด รีจิสเตอร์ดังกล่าวเรียกว่าพอร์ต ใน Intel 8088 มีรูปแบบคำสั่งสองแบบ รูปแบบหนึ่ง Opcode 8 บิตระบุการดำเนินการ I/O ตามด้วยที่อยู่พอร์ต 8 บิต Opcode I/O อื่น ๆ หมายความว่าที่อยู่พอร์ตอยู่ในรีจิสเตอร์ DX 16 บิต 8088 สามารถระบุที่อยู่พอร์ตได้กี่พอร์ตในแต่ละโหมดการระบุที่อยู่

**7.2** รูปแบบคำสั่งที่คล้ายกันถูกใช้ในตระกูลไมโครโปรเซสเซอร์ Zilog Z8000 ในกรณีนี้มีความสามารถในการระบุที่อยู่พอร์ตโดยตรง ซึ่งที่อยู่พอร์ต 16 บิตเป็นส่วนหนึ่งของคำสั่ง และความสามารถในการระบุที่อยู่พอร์ตโดยอ้อม ซึ่งคำสั่งอ้างถึงรีจิสเตอร์อเนกประสงค์ 16 บิตตัวใดตัวหนึ่งที่มีที่อยู่พอร์ต Z8000 สามารถระบุที่อยู่พอร์ตได้กี่พอร์ตในแต่ละโหมดการระบุที่อยู่ I/O

**7.3** Z8000 ยังรวมความสามารถในการถ่ายโอน I/O แบบบล็อก ซึ่งต่างจาก DMA อยู่ภายใต้การควบคุมโดยตรงของโปรเซสเซอร์ คำสั่งถ่ายโอนบล็อกระบุรีจิสเตอร์ที่อยู่พอร์ต (Rp) รีจิสเตอร์นับ (Rc) และรีจิสเตอร์ปลายทาง (Rd) Rd มีที่อยู่หน่วยความจำหลักที่ไบต์แรกที่อ่านจากพอร์ตอินพุตจะถูกเก็บ Rc คือรีจิสเตอร์อเนกประสงค์ 16 บิตใด ๆ สามารถถ่ายโอนบล็อกข้อมูลขนาดใหญ่แค่ไหน

**7.4** พิจารณาไมโครโปรเซสเซอร์ที่มีคำสั่งถ่ายโอน I/O แบบบล็อกเช่นที่พบใน Z8000 หลังจากดำเนินการครั้งแรกแล้ว คำสั่งดังกล่าวใช้รอบนาฬิกาห้ารอบในการดำเนินการซ้ำ อย่างไรก็ตาม หากใช้คำสั่ง I/O แบบไม่ปิดกั้น (Non-Blocking) จะต้องใช้รอบนาฬิกา 20 รอบสำหรับการดึงและดำเนินการ คำนวณความเพิ่มขึ้นของความเร็วด้วยคำสั่ง I/O แบบบล็อกเมื่อถ่ายโอนบล็อก 128 ไบต์

**7.5** ระบบอิงจากไมโครโปรเซสเซอร์ 8 บิตและมีอุปกรณ์ I/O สองชิ้น ตัวควบคุม I/O สำหรับระบบนี้ใช้รีจิสเตอร์ควบคุมและสถานะแยกกัน อุปกรณ์ทั้งสองรับส่งข้อมูลทีละหนึ่งไบต์ อุปกรณ์แรกมีสองสายสถานะและสามสายควบคุม อุปกรณ์ที่สองมีสามสายสถานะและสี่สายควบคุม

- (ก) ต้องการรีจิสเตอร์โมดูลควบคุม I/O ขนาด 8 บิตกี่ตัวสำหรับการอ่านสถานะและควบคุมแต่ละอุปกรณ์
- (ข) จำนวนรีจิสเตอร์โมดูลควบคุมที่ต้องการทั้งหมดคือเท่าใด กำหนดว่าอุปกรณ์ที่สองเป็นอุปกรณ์ส่งออกเท่านั้น
- (ค) ต้องการที่อยู่เฉพาะกี่ที่อยู่เพื่อควบคุมอุปกรณ์ทั้งสอง

**7.6** สำหรับ Programmed I/O รูปที่ 7.5 บ่งชี้ว่าโปรเซสเซอร์ติดอยู่ในการรอและตรวจสอบสถานะของอุปกรณ์ I/O เพื่อเพิ่มประสิทธิภาพ ซอฟต์แวร์ I/O สามารถเขียนเพื่อให้โปรเซสเซอร์ตรวจสอบสถานะของอุปกรณ์เป็นระยะ หากอุปกรณ์ไม่พร้อม โปรเซสเซอร์สามารถข้ามไปยังงานอื่น หลังจากช่วงเวลาที่กำหนด โปรเซสเซอร์จะกลับมาตรวจสอบสถานะอีกครั้ง

- (ก) พิจารณาโครงร่างข้างต้นสำหรับการส่งออกข้อมูลทีละอักขระไปยังเครื่องพิมพ์ที่ทำงานที่ 10 ตัวอักษรต่อวินาที (cps) จะเกิดอะไรขึ้นหากสถานะถูกสแกนทุก 200 ms
- (ข) พิจารณาแป้นพิมพ์ที่มีบัฟเฟอร์อักขระเดียว โดยเฉลี่ยอักขระถูกป้อนในอัตรา 10 cps อย่างไรก็ตาม ช่วงเวลาระหว่างการกดแป้นสองครั้งติดต่อกันอาจสั้นเพียง 60 ms โปรแกรม I/O ควรสแกนแป้นพิมพ์ด้วยความถี่เท่าใด

**7.7** ไมโครโปรเซสเซอร์สแกนสถานะของอุปกรณ์ส่งออก I/O ทุก 20 ms ซึ่งทำได้โดยใช้ตัวตั้งเวลาแจ้งเตือนโปรเซสเซอร์ทุก 20 ms อินเทอร์เฟซของอุปกรณ์ประกอบด้วยสองพอร์ต: หนึ่งสำหรับสถานะและหนึ่งสำหรับส่งออกข้อมูล ใช้เวลานานแค่ไหนในการสแกนและให้บริการอุปกรณ์ กำหนดอัตรานาฬิกา 8 MHz สมมติว่าคำสั่งที่เกี่ยวข้องทั้งหมดใช้เวลา 12 รอบนาฬิกา

**7.8** ในส่วนที่ 7.3 มีการระบุข้อดีหนึ่งข้อและข้อเสียหนึ่งข้อของ Memory-Mapped I/O เปรียบเทียบกับ Isolated I/O จงระบุข้อดีสองข้อเพิ่มเติมและข้อเสียสองข้อเพิ่มเติม

**7.9** ระบบเฉพาะถูกควบคุมโดยผู้ปฏิบัติงานผ่านคำสั่งที่ป้อนจากแป้นพิมพ์ จำนวนคำสั่งเฉลี่ยที่ป้อนในช่วง 8 ชั่วโมงคือ 60 คำสั่ง

- (ก) สมมติว่าโปรเซสเซอร์สแกนแป้นพิมพ์ทุก 100 ms แป้นพิมพ์จะถูกตรวจสอบกี่ครั้งในช่วง 8 ชั่วโมง
- (ข) จำนวนครั้งที่โปรเซสเซอร์เข้าชมแป้นพิมพ์จะลดลงเท่าใดหากใช้ Interrupt-Driven I/O

**7.10** สมมติว่า 8255A ที่แสดงในรูปที่ 7.9 ถูกกำหนดค่าดังนี้: พอร์ต A เป็นอินพุต พอร์ต B เป็นเอาต์พุต และบิตทั้งหมดของพอร์ต C เป็นเอาต์พุต จงแสดงบิตของรีจิสเตอร์ควบคุมเพื่อกำหนดการกำหนดค่านี้

**7.11** พิจารณาระบบที่ใช้ Interrupt-Driven I/O สำหรับอุปกรณ์เฉพาะที่ถ่ายโอนข้อมูลโดยเฉลี่ย 8 KB/s อย่างต่อเนื่อง

- (ก) สมมติว่าการประมวลผลอินเทอร์รัปต์ใช้เวลาประมาณ 100 µs (เวลาในการข้ามไปยัง ISR ดำเนินการ และกลับมาที่โปรแกรมหลัก) กำหนดเศษส่วนของเวลาโปรเซสเซอร์ที่ถูกใช้โดยอุปกรณ์ I/O นี้เมื่ออินเทอร์รัปต์ทุกไบต์
- (ข) สมมติว่าอุปกรณ์มีบัฟเฟอร์ 16 ไบต์สองตัว และขัดจังหวะโปรเซสเซอร์เมื่อบัฟเฟอร์ใดบัฟเฟอร์หนึ่งเต็ม การประมวลผลอินเทอร์รัปต์ใช้เวลานานกว่าเนื่องจาก ISR ต้องถ่ายโอน 16 ไบต์ ขณะดำเนินการ ISR โปรเซสเซอร์ใช้เวลาประมาณ 8 µs สำหรับการถ่ายโอนแต่ละไบต์ กำหนดเศษส่วนของเวลาโปรเซสเซอร์ที่ถูกใช้โดยอุปกรณ์ I/O นี้
- (ค) สมมติว่าโปรเซสเซอร์ติดตั้งคำสั่งถ่ายโอน I/O แบบบล็อก ซึ่งอนุญาตให้ ISR ที่เกี่ยวข้องถ่ายโอนแต่ละไบต์ของบล็อกได้เพียง 2 µs กำหนดเศษส่วนของเวลาโปรเซสเซอร์ที่ถูกใช้โดยอุปกรณ์ I/O นี้

**7.12** ในระบบทั้งหมดที่มีโมดูล DMA แทบทุกระบบ DMA ไปยังหน่วยความจำหลักจะได้รับลำดับความสำคัญสูงกว่าการเข้าถึงหน่วยความจำหลักของ CPU เพราะเหตุใด

**7.13** โมดูล DMA กำลังถ่ายโอนอักขระไปยังหน่วยความจำโดยใช้ Cycle Stealing จากอุปกรณ์ที่ส่งที่ 9,600 bps โปรเซสเซอร์กำลังดึงคำสั่งในอัตรา 1 ล้านคำสั่งต่อวินาที (1 MIPS) โปรเซสเซอร์จะช้าลงเท่าใดเนื่องจากกิจกรรม DMA

**7.14** พิจารณาระบบที่รอบบัสใช้เวลา 500 ns การถ่ายโอนการควบคุมบัสในแต่ละทิศทาง จากโปรเซสเซอร์ไปยังอุปกรณ์ I/O หรือในทางกลับกัน ใช้เวลา 250 ns อุปกรณ์หนึ่งชิ้นมีอัตราการถ่ายโอนข้อมูล 50 KB/s และใช้ DMA ข้อมูลถูกถ่ายโอนทีละไบต์

- (ก) สมมติว่าใช้ DMA ในโหมด Burst นั่นคืออินเทอร์เฟซ DMA ได้รับความเป็นเจ้าของก่อนเริ่มการถ่ายโอนบล็อกและคงการควบคุมบัสจนกว่าบล็อกทั้งหมดจะถูกถ่ายโอน อุปกรณ์จะผูกมัดบัสนานแค่ไหนเมื่อถ่ายโอนบล็อก 128 ไบต์
- (ข) ทำซ้ำการคำนวณสำหรับโหมด Cycle-Stealing

**7.15** การตรวจสอบไดอะแกรมเวลาของ 8237A บ่งชี้ว่าเมื่อการถ่ายโอนบล็อกเริ่มต้น จะต้องใช้สามรอบนาฬิกาบัสต่อรอบ DMA ในระหว่างรอบ DMA 8237A จะถ่ายโอนข้อมูลหนึ่งไบต์ระหว่างหน่วยความจำกับอุปกรณ์ I/O

- (ก) สมมติว่าใช้ 8237A ที่อัตรา 5 MHz ใช้เวลานานแค่ไหนในการถ่ายโอนหนึ่งไบต์
- (ข) อัตราการถ่ายโอนข้อมูลสูงสุดที่บรรลุได้คือเท่าใด
- (ค) สมมติว่าหน่วยความจำไม่เร็วพอและต้องแทรกสองรอบรอต่อรอบ DMA อัตราการถ่ายโอนข้อมูลจริงจะเป็นเท่าใด

**7.16** สมมติว่าในระบบของปัญหาก่อนหน้า รอบหน่วยความจำใช้เวลา 750 ns สามารถลดอัตรานาฬิกาของบัสได้ถึงค่าใดโดยไม่ส่งผลต่ออัตราการถ่ายโอนข้อมูลที่บรรลุได้

**7.17** ตัวควบคุม DMA ให้บริการลิงค์โทรคมนาคมแบบรับอย่างเดียวสี่เส้น (หนึ่งเส้นต่อช่อง DMA) ที่ความเร็ว 64 Kbps แต่ละเส้น

- (ก) ควรดำเนินการตัวควบคุมในโหมด Burst หรือโหมด Cycle-Stealing
- (ข) ควรใช้โครงร่างลำดับความสำคัญใดสำหรับการให้บริการช่อง DMA

**7.18** คอมพิวเตอร์ 32 บิตมีสอง Selector Channel และหนึ่ง Multiplexor Channel แต่ละ Selector Channel รองรับดิสก์แม่เหล็กสองตัวและเทปแม่เหล็กสองตัว Multiplexor Channel มีเครื่องพิมพ์บรรทัดสองเครื่อง เครื่องอ่านการ์ดสองเครื่อง และเทอร์มินัล VDT สิบตัวที่เชื่อมต่ออยู่ สมมติอัตราการถ่ายโอนต่อไปนี้:

| อุปกรณ์ | อัตรา |
|---|---|
| ดิสก์ไดรฟ์ | 800 Kbytes/s |
| เทปแม่เหล็กไดรฟ์ | 200 Kbytes/s |
| เครื่องพิมพ์บรรทัด | 6.6 Kbytes/s |
| เครื่องอ่านการ์ด | 1.2 Kbytes/s |
| VDT | 1 Kbyte/s |

ประมาณอัตราการถ่ายโอน I/O รวมสูงสุดในระบบนี้

**7.19** คอมพิวเตอร์ประกอบด้วยโปรเซสเซอร์และอุปกรณ์ I/O D ที่เชื่อมต่อกับหน่วยความจำหลัก M ผ่านบัสที่ใช้ร่วมกันที่มีความกว้างบัสข้อมูลหนึ่งคำ โปรเซสเซอร์สามารถดำเนินการคำสั่งสูงสุด 10⁶ คำสั่งต่อวินาที คำสั่งเฉลี่ยต้องการห้ารอบเครื่อง สามในจำนวนนั้นใช้บัสหน่วยความจำ การดำเนินการอ่านหรือเขียนหน่วยความจำใช้หนึ่งรอบเครื่อง สมมติว่าโปรเซสเซอร์กำลังดำเนินการโปรแกรม "Background" อย่างต่อเนื่องที่ต้องการ 95% ของอัตราการดำเนินการคำสั่ง แต่ไม่มีคำสั่ง I/O ใด ๆ สมมติว่าหนึ่งรอบโปรเซสเซอร์เท่ากับหนึ่งรอบบัส

- (ก) หากใช้ Programmed I/O และการถ่ายโอน I/O หนึ่งคำแต่ละครั้งต้องการให้โปรเซสเซอร์ดำเนินการสองคำสั่ง ประมาณอัตราการถ่ายโอนข้อมูล I/O สูงสุดเป็นคำต่อวินาทีที่เป็นไปได้ผ่าน D
- (ข) ประมาณอัตราเดียวกันหากใช้ DMA

**7.20** แหล่งข้อมูลผลิตอักขระ IRA 7 บิต โดยแต่ละตัวต่อท้ายด้วยบิตพาริตี จงหาสูตรสำหรับอัตราข้อมูลที่มีประสิทธิภาพสูงสุด (อัตรา IRA data bits) ผ่านสายขนาด R bps สำหรับกรณีต่อไปนี้:

- (ก) การส่งแบบอะซิงโครนัส ที่มี Stop Bit 1.5 หน่วย
- (ข) การส่งแบบ Bit-Synchronous ที่มีเฟรมประกอบด้วยบิตควบคุม 48 บิตและข้อมูล 128 บิต
- (ค) เหมือนกับ (ข) แต่มีฟิลด์ข้อมูล 1,024 บิต
- (ง) แบบ Character-Synchronous ที่มีเก้าอักขระควบคุมต่อเฟรมและ 16 อักขระข้อมูล
- (จ) เหมือนกับ (ง) แต่มี 128 อักขระข้อมูล

**7.21** ผู้หญิงสองคนอยู่คนละฝั่งของรั้วสูง คนหนึ่งชื่อ Apple-Server มีต้นแอปเปิลที่สวยงามที่กำลังผลิตแอปเปิลอร่อยๆ บนฝั่งของเธอ เธอยินดีจะส่งแอปเปิลให้กับอีกคนเมื่อต้องการ อีกคนชื่อ Apple-Eater ชอบกินแอปเปิลแต่ไม่มีแอปเปิล เธอต้องกินแอปเปิลในอัตราคงที่ ไม่สามารถพูดคุยกันได้ และปัญหาคือต้องส่งแอปเปิลจาก Apple-Server ไปยัง Apple-Eater ในอัตราที่ถูกต้อง

- (ก) สมมติว่ามีนาฬิกาปลุกวางอยู่บนรั้วและนาฬิกาสามารถตั้งปลุกได้หลายครั้ง นาฬิกาสามารถใช้แก้ปัญหาได้อย่างไร จงวาดไดอะแกรมเวลาเพื่อแสดงให้เห็นวิธีแก้ไข
- (ข) สมมติว่าไม่มีนาฬิกาปลุก แต่ Apple-Eater มีธงที่สามารถโบกได้เมื่อต้องการแอปเปิล เสนอวิธีแก้ใหม่ การที่ Apple-Server มีธงด้วยจะเป็นประโยชน์หรือไม่ หากเป็นเช่นนั้น รวมสิ่งนี้ไว้ในวิธีแก้ไข อภิปรายข้อเสียของแนวทางนี้
- (ค) นำธงออกและสมมติว่ามีเชือกยาว เสนอวิธีแก้ไขที่ดีกว่า (ข) โดยใช้เชือก

**7.22** สมมติว่าไมโครโปรเซสเซอร์ 16 บิตหนึ่งตัวและไมโครโปรเซสเซอร์ 8 บิตสองตัวจะถูกเชื่อมต่อกับบัสระบบ โดยมีรายละเอียดดังต่อไปนี้:

1. ไมโครโปรเซสเซอร์ทั้งหมดมีคุณลักษณะฮาร์ดแวร์ที่จำเป็นสำหรับการถ่ายโอนทุกประเภท ได้แก่ Programmed I/O, Interrupt-Driven I/O และ DMA
2. ไมโครโปรเซสเซอร์ทั้งหมดมีบัสที่อยู่ 16 บิต
3. เมมโมรีบอร์ดสองแผ่นแต่ละแผ่นมีความจุ 64 Kbytes ถูกเชื่อมต่อกับบัส นักออกแบบต้องการใช้หน่วยความจำที่ใช้ร่วมกันที่ใหญ่ที่สุดเท่าที่จะเป็นไปได้
4. บัสระบบรองรับสายอินเทอร์รัปต์สูงสุดสี่เส้นและสายDMA หนึ่งเส้น

กำหนดสมมติฐานที่จำเป็นอื่น ๆ และ:
- (ก) ให้ข้อกำหนดบัสระบบในแง่จำนวนและประเภทของสาย
- (ข) อธิบายโปรโตคอลที่เป็นไปได้สำหรับการสื่อสารบนบัส (ลำดับการอ่าน-เขียน อินเทอร์รัปต์ และ DMA)
- (ค) อธิบายว่าอุปกรณ์ที่กล่าวถึงข้างต้นเชื่อมต่อกับบัสระบบได้อย่างไร
