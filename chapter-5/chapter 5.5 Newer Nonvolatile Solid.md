🔹 5.5 New Nonvolatile Memory
🧠 โครงสร้าง memory hierarchy
SRAM → เร็วสุด แพง
DRAM → หลัก
Disk → ช้า แต่ถูก
Flash → อยู่ตรงกลาง

🔸 เทคโนโลยีใหม่
| เทคโนโลยี | จุดเด่น                | การใช้งาน           |
| --------- | ---------------------- | ------------------- |
| STT-RAM   | เร็วมาก + ไม่ลบข้อมูล  | cache / main memory |
| PCRAM     | ใช้ phase เปลี่ยนสถานะ | แทน DRAM            |
| ReRAM     | ใช้ความต้านทาน         | แทน storage + RAM   |

🔸 แนวโน้ม
Flash ยังครองตลาด
