🔹 5.4 Flash Memory
🧠 คุณสมบัติ
เป็น non-volatile memory
เก็บข้อมูลได้แม้ไม่มีไฟ
🔸 หลักการทำงาน
ใช้ floating gate transistor
เก็บ electron → แทนค่า 0/1

🔸 เปรียบเทียบ NOR vs NAND
| คุณสมบัติ     | NOR           | NAND               |
| ------------- | ------------- | ------------------ |
| การเข้าถึง    | random access | block access       |
| ความเร็วอ่าน  | เร็ว          | ปานกลาง            |
| ความเร็วเขียน | ช้า           | เร็ว               |
| ความหนาแน่น   | ต่ำ           | สูง                |
| การใช้งาน     | embedded      | storage (USB, SSD) |
